# 📈 比特币价格预测与时间序列建模

## 📌 1. 项目背景

比特币价格具有高波动性，对其进行预测有助于投资决策与风险控制。本项目基于比特币美元价格时间序列分析方法，对比特币价格进行建模与预测，并评估不同模型的预测性能。

---

## 🤖 2. 数据来源

- 数据来源：Yahoo Finance  
- 标的：BTC-USD 收盘价
- 时间范围：2020-01-01 至 2025-12-31  
- 数据频率：日度数据

---

## 📉 3. 方法与步骤

- 1.数据获取与可视化

- 2.平稳性检验并确定差分阶数（KPSS检验）
     
- 3.模型选择

- 4.预测数据分类

- 5.模型评估（MAE、RMSE）

- 6.与 Holt-Winters 模型对比

---

## 🎯 4. 结果
- 比特币价格具有明显趋势性与高波动性  
- ARIMA模型能够提供稳定预测结果
- Holt-Winters适合短期波动  
- 使用RMSE和MAE选择最优模型  

---

## 📊 5. 商业价值

- 支持加密货币投资决策  
- 帮助识别趋势与波动  
- 可用于量化交易策略  

---

## 💻 6. 工具

- R语言  
- 时间序列分析  
- ARIMA（自回归积分滑动平均模型）
- Holt-Winters（指数平滑方法）
